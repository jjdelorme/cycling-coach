#### Step 1.A: Heart Rate Physiological Filters (New branch in `clean_ride_data`)
1.  **The Unit Test Harness:** Define the verification requirement in `tests/test_metrics.py`.
    *   *Test Cases to Write:*
        *   `test_clean_ride_data_hr_bounds`: Assert that HR values < 30 or > 240 are zeroed or interpolated out. (e.g., input `[120, 125, 255, 125, 120]`, expect the `255` to be removed/smoothed).
        *   `test_clean_ride_data_hr_roc`: Assert that an impossible sudden jump in HR (e.g., +20 bpm in 1 second) is flagged and interpolated. (e.g., input `[120, 121, 145, 121, 120]`, expect the `145` anomaly to be smoothed).
2.  **The Implementation:** Modify `clean_ride_data` in `server/metrics.py` to handle HR uniquely. Replace the generic `_clean_single_array` call for HR with a new `_clean_hr_array` function:
    ```python
    def _clean_hr_array(arr):
        if arr is None or len(arr) == 0:
            return arr
        arr = np.array([float(x) if x is not None else np.nan for x in arr], dtype=float)
        
        # 1. Physiological Bounds check (Humans alive and exercising)
        arr[(arr < 30) | (arr > 240)] = np.nan
        
        # 2. Rate of change (RoC) filter (max 10 bpm per sec is physiologically impossible)
        if len(arr) > 1:
            diffs = np.abs(np.diff(arr))
            # diffs[i] is arr[i+1] - arr[i]. If diff > 10, mark arr[i+1] as nan
            roc_mask = np.concatenate(([False], diffs > 10.0))
            arr[roc_mask] = np.nan
        
        # 3. Interpolate the NaNs that were just created
        nans = np.isnan(arr)
        if np.any(nans) and np.any(~nans):
            x = np.arange(len(arr))
            mask = ~nans
            if np.sum(mask) >= 2:
                from scipy.interpolate import interp1d
                from itertools import groupby
                f = interp1d(x[mask], arr[mask], kind='linear', bounds_error=False, fill_value="extrapolate")
                # Only interpolate small gaps (<10s) just like we do for power
                for is_nan, group in groupby(enumerate(nans), key=lambda x: x[1]):
                    group_list = list(group)
                    if is_nan and len(group_list) < 10:
                        indices = [i for i, _ in group_list]
                        arr[indices] = f(indices)
                        
        return arr
    ```

#### Step 1.B: Power Rolling Z-Score Outlier Detection
1.  **The Unit Test Harness:** Define the verification requirement in `tests/test_metrics.py`.
    *   *Test Cases to Write:*
        *   `test_clean_ride_data_power_zscore`: Create an array with a steady baseline (e.g., 200W) and insert a massive brief spike (e.g., 1000W) surrounded by tight noise (std dev ~ 5W). The rolling Z-score should flag the 1000W spike (z > 4) and replace it with the rolling mean. 
        *   Ensure a sustained 1000W effort (e.g., 15s steady sprint) is NOT filtered out, proving the rolling window adapts standard deviations appropriately.
2.  **The Implementation:** Update `_clean_single_array` when `is_power=True` to apply a rolling Z-score. Add import: `from scipy.ndimage import uniform_filter1d`.
    ```python
    # Inside _clean_single_array for is_power=True (replace the simple median filter section):
    if is_power and len(arr) >= 30:
        # 1. Hard cutoff (Impossible human output)
        arr[arr > 2500] = 0.0
        
        # 2. Rolling Z-Score Outlier Detection
        window = 15 # 15-second rolling window
        
        # Temporarily fill NaNs for rolling calculations to prevent NaN propagation
        temp_arr = arr.copy()
        nans_temp = np.isnan(temp_arr)
        if np.any(nans_temp) and np.any(~nans_temp):
            from scipy.interpolate import interp1d
            x = np.arange(len(temp_arr))
            mask = ~nans_temp
            f_fill = interp1d(x[mask], temp_arr[mask], kind='nearest', bounds_error=False, fill_value="extrapolate")
            temp_arr[nans_temp] = f_fill(x[nans_temp])
            
        from scipy.ndimage import uniform_filter1d
        rolling_mean = uniform_filter1d(temp_arr, size=window)
        rolling_sq_mean = uniform_filter1d(temp_arr**2, size=window)
        
        # Calculate standard deviation safely to avoid negative variances due to float precision
        variance = np.maximum(rolling_sq_mean - rolling_mean**2, 0)
        rolling_std = np.sqrt(variance)
        
        # Avoid division by zero when signal is perfectly steady
        rolling_std[rolling_std < 1.0] = 1.0
        
        # Compute Z-score contextual to the current 15-second window
        z_scores = np.abs((temp_arr - rolling_mean) / rolling_std)
        
        # Flag anomalies (Z > 4.0 captures sensor glitches but ignores organic sprint onset)
        anomalies = z_scores > 4.0
        
        # Replace anomalies with the contextual rolling mean
        if np.any(anomalies):
            arr[anomalies] = rolling_mean[anomalies]
    ```
