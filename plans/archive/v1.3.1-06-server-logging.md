# Plan: Server Logging Improvements

**Version:** 1.3.1
**Type:** Infrastructure
**Complexity:** Low-Medium

## Summary

Logging infrastructure is essentially non-existent. No `basicConfig()` call means existing `logger.info()` calls are silently dropped. `ingest.py` uses raw `print()`. No query timing, no request middleware.

## Current State

- No logging configuration anywhere
- `sync.py` ~20 log calls at INFO — silently dropped
- `intervals_icu.py` ~6 log calls — silently dropped
- `ingest.py` uses 15 `print()` statements
- `database.py` has no logging
- No query timing, no structured format, no request middleware

## Phase 3: Database Performance Monitoring (~2-3h)

**File: `server/database.py`**

Add at top:
```python
import logging, time
logger = logging.getLogger(__name__)
SLOW_QUERY_MS = int(os.environ.get("SLOW_QUERY_MS", "100"))
```

### Modify `_DbConnection.execute()`

```python
def execute(self, sql, params=None):
    adapted = self._adapt_sql(sql)
    t0 = time.monotonic()
    self._cursor.execute(adapted, params)
    elapsed_ms = (time.monotonic() - t0) * 1000
    if elapsed_ms >= SLOW_QUERY_MS:
        sql_preview = adapted[:200] + ("..." if len(adapted) > 200 else "")
        logger.warning("Slow query (%.1fms): %s", elapsed_ms, sql_preview)
    return self._cursor
```

### Modify `_DbConnection.executemany()`

Same timing pattern. Log slow operations and always log bulk ops with 100+ rows at DEBUG.

**Sensitive data:** Log SQL template only (contains `%s` placeholders), never parameter values. Truncate to 200 chars.

## Phase 4: Request Middleware (~30m)

**File: `server/main.py`**

```python
import time
from starlette.middleware.base import BaseHTTPMiddleware

class RequestLoggingMiddleware(BaseHTTPMiddleware):
    async def dispatch(self, request, call_next):
        t0 = time.monotonic()
        response = await call_next(request)
        elapsed_ms = (time.monotonic() - t0) * 1000
        path = request.url.path
        if not path.startswith("/assets/"):
            logger.info("%s %s %d %.0fms", request.method, path, response.status_code, elapsed_ms)
        return response

app.add_middleware(RequestLoggingMiddleware)
```

Place after `CORSMiddleware`. No correlation IDs — single-user app doesn't need them.
