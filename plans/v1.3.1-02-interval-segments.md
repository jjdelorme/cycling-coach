# Plan: Interval Segments Import

**Version:** 1.3.1
**Type:** Bug/Gap
**Complexity:** Low-Medium

## Summary

The `lap` array in source JSON files is completely ignored during ingest. 91 of 291 rides (31%) have multiple laps with rich interval-level metrics. Import this data and expose it via API.

## Current State

- `parse_ride_json()` in `server/ingest.py` extracts session, sport, user_profile, and record but never touches `data.get("lap", [])`
- No `ride_laps` table exists
- Data distribution: 7 rides have 0 laps, 193 have 1 lap, 91 have multiple (up to 46)
- Intensity values observed: `None`, `"active"`, `5` (numeric — must coerce to None)
- Lap trigger values observed: `"time"`, `"distance"`, `"session_end"`, `21` (numeric — must coerce to None), `"manual"`
- GPS coordinates use Garmin semicircle format

## Phase 2: Ingest

### 2.1 Add GPS helper to `server/ingest.py`

```python
def _semicircles_to_degrees(val):
    """Convert Garmin semicircle coordinates to decimal degrees, or return None."""
    if val is None:
        return None
    if abs(val) > 180:
        return val * (180 / 2**31)
    return val
```

Refactor existing GPS conversion in `parse_ride_json` to use this helper.

### 2.4 Add `backfill_laps()` function

```python
def backfill_laps(conn, rides_dir=None):
    """Backfill lap data for rides ingested before lap support."""
    rides_with_laps = set(r["ride_id"] for r in conn.execute("SELECT DISTINCT ride_id FROM ride_laps").fetchall())
    all_rides = conn.execute("SELECT id, filename FROM rides").fetchall()
    backfilled = 0
    for ride_row in all_rides:
        if ride_row["id"] in rides_with_laps:
            continue
        filepath = os.path.join(rides_dir or RIDES_DIR, ride_row["filename"])
        if not os.path.exists(filepath):
            continue
        _ride, _records, _power_bests, laps = parse_ride_json(filepath)
        if not laps:
            continue
        conn.executemany(...)  # same INSERT as above
        backfilled += 1
    print(f"Backfilled laps for {backfilled} rides")
    return backfilled
```
