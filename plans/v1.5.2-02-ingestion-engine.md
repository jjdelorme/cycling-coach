## 📝 Step-by-Step Implementation Details

### Phase 1: Statistical Polish
1.  **Step 1.A (Power Z-Score):**
    *   *Target File:* `server/metrics.py` -> `clean_ride_data`
    *   *Change:* Implement a rolling 60s Z-score check. If `|Z| > 3.5`, flag as an outlier and interpolate. This is more robust than the current static 2500W cap.
2.  **Step 1.B (HR Physiological Filters):**
    *   *Target File:* `server/metrics.py` -> `clean_ride_data`
    *   *Change:* 
        *   Mask HR values `< 40` or `> 220`.
        *   Mask HR spikes where `|diff| > 20 bpm/sec` (impossible acceleration).
        *   Interpolate small gaps created by masking.
